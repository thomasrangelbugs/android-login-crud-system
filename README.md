# Android Login + CRUD System

> Sistema Android com login, SQLite e CRUD de pessoas.

## Sobre o projeto

App LoginSistema com autenticacao local (usuarios pre-cadastrados), CRUD completo de pessoas persistido em SQLite, validacao de CPF, mascaras de CPF e CEP, busca dinamica e RecyclerView com acoes de editar e excluir via AlertDialog.

## Funcionalidades principais

- Login local com usuarios pre-cadastrados
- CRUD completo de pessoas em SQLite
- Validacao de CPF e mascaras CPF/CEP
- Busca dinamica em tempo real
- RecyclerView com editar e excluir
- Material Design e AlertDialog

## Tecnologias utilizadas

- Android Java
- SQLite
- RecyclerView
- Material Design
- Gradle

## Como executar

Abra a pasta no Android Studio, sincronize Gradle e execute.

## Deploy / Demonstracao

App Android - gere APK via Android Studio.
